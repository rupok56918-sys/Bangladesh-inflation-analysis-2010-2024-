# Bangladesh Inflation Analysis

This project analyzes inflation trends in Bangladesh using Microsoft Excel and a research report prepared in Microsoft Word.

## Tools Used:
- Microsoft Excel (Data analysis, mean, variance, standard deviation, graphs)
- Microsoft Word (Report writing)

## Description:
This project is part of a statistical analysis of inflation data in Bangladesh.
